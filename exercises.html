Exercises

These are the exercises that i did in class

First of all i had to take a Safety quiz which allowed me access to the resources that the FABLAB had to offer

My first hands on activity was to use Autodesk fusion 360 to create a keychain.

After creating the keychain on autodesk,i had to bring the file into ultimaker cura in order to 3d print the keychain.These are the print settings that i used to make the keychain come out smoothly.

Overall im happy with the end result.

My second assignment was to test the design rules for a 3D printer: bridging, overhang, infill density, clearance, etc . My model included supports and overhangs because of the angle exceed 45 degrees and the machine could not print on air

The last exercise which i did involved programming and electronics.I First had to use tinkercad to create a simulation of the actual arduino board

After which I connected the wires and electronics to an actual arduino board and tested out the programming.
